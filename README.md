##  About the Project

This project is a simple yet polished implementation of the classic Tic Tac Toe game. It demonstrates real-time user interaction, dynamic UI updates, and core game logic using pure JavaScript.

The game supports two players, automatically detects winners, and handles draw scenarios with a clean user experience.

---

##  Features

- Two-player gameplay (X and O)
- Automatic win detection
- Draw detection logic
- Reset and New Game functionality
- Clean and responsive user interface
